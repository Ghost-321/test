# Winter School Templates

Eleven plain HTML, CSS, and light JavaScript template sets for beginner students.

## What changed in this version
The project templates now mimic recognisable real-world website patterns without copying any one brand exactly:
- `normal-site-template-a`: public service / information website pattern.
- `normal-site-template-b`: editorial storytelling / magazine pattern.
- `normal-site-template-c`: sports scores / FIFA website pattern with left navigation.
- `normal-site-template-d`: domestic violence awareness / support pattern with top navigation.
- `normal-site-template-e`: pregnancy information pattern with left navigation.
- `normal-site-template-f`: bold generic website pattern with left navigation.
- `normal-site-template-g`: top navigation health pattern with DV support, pregnancy infographic, and assistance form designs.
- `normal-site-template-h`: pregnancy information pattern with top navigation.
- `shop-site-template`: modern ecommerce / product grid pattern.
- `shop-site-template-sidebar`: ecommerce / product pattern with left navigation.
- `html-css-learning-starter`: a teaching starter that explains those patterns.
