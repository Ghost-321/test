# Pregnancy Guide

Pregnancy information website template using left navigation.

## Navigation teaching note
This template uses a sidebar navigation. The current page is shown with `class="active"` on that page link. The mobile menu works because `js/script.js` toggles the `.is-open` class on `.nav-links`.

## Pages included
- index.html
- stages.html
- health.html
- appointments.html
- resources.html
- contact.html
- student-1.html
- student-2.html
- student-3.html
- student-4.html

## Student reminder
This is not medical advice. Students should use trusted sources, write in general language, and encourage readers to contact qualified health professionals.
