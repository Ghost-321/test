# Safe Support

Domestic violence awareness and support website template using top navigation.

## Navigation teaching note
This template uses a horizontal top navigation. The current page is shown with `class="active"` on that page link. The mobile menu works because `js/script.js` toggles the `.is-open` class on `.nav-links`.

## Pages included
- index.html
- safety.html
- support.html
- resources.html
- contact.html
- student-1.html
- student-2.html
- student-3.html
- student-4.html

## Student reminder
For sensitive health topics, use respectful wording and link to trusted sources. Replace placeholder support information with verified local services.
