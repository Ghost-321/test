# Pregnancy Guide Top Nav

Pregnancy information website template using navigation across the top.

## Navigation teaching note
This is the top-navigation version of template E. The selected page is shown with `class="active"` on that page link. On small screens, `js/script.js` toggles `.is-open` so the menu opens and closes.

## Pages included
- index.html
- stages.html
- health.html
- appointments.html
- resources.html
- contact.html
- student-1.html
- student-2.html
- student-3.html
- student-4.html

## Student reminder
This is not medical advice. Students should use trusted sources, write in general language, and encourage readers to contact qualified health professionals.
