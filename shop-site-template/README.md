# Market Studio

## What this template is for
A realistic ecommerce prototype inspired by modern shop themes: product grids, categories, featured products, and a visual cart demo only.

## Pages included
- index.html
- about.html
- shop.html
- product.html
- cart.html
- faq.html
- contact.html
- student-1.html
- student-2.html
- student-3.html
- student-4.html

## How to customise
- Change the logo in the shared header area of each HTML file.
- Change colours in `css/style.css` using the variables at the top.
- Replace placeholder boxes with images from `assets/images`.
- Keep useful alt text, labels, headings, and readable contrast.

## Student pages
Each student should edit one page from `student-1.html` to `student-4.html`.

## 4-day workshop idea
- Day 1: Choose a topic and edit headings, colours, and navigation.
- Day 2: Add realistic content, cards, images, and student pages.
- Day 3: Improve resources, FAQ, product/story details, and accessibility.
- Day 4: Test on mobile, polish, and present.

## Extension challenges
- Add a new page.
- Add a filter, gallery, task list, or timeline.
- Improve the footer and contact page.
- Add a small JavaScript interaction.
