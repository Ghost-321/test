# Match Centre

## What this template is for
A regular sports website template with a left navigation, scores, FIFA content, team pages, contact page, and student pages.

## Pages included
- index.html
- scores.html
- fifa.html
- teams.html
- contact.html
- student-1.html
- student-2.html
- student-3.html
- student-4.html

## How to customise
- Change the site name in the left navigation.
- Edit the selected page by moving the `active` class to the correct navigation link.
- Change colours in `css/style.css` using the variables at the top.
- Replace score placeholders with real or imaginary match results.
- Add images inside `assets/images`.

## Student pages
Each student should edit one page from `student-1.html` to `student-4.html`.
