Image folder guide
==================

Add your own images for the Field Notes template here.

Suggested files:
- logo-placeholder.png
- hero-image.jpg
- card-image-1.jpg, card-image-2.jpg, card-image-3.jpg
- student-1.jpg, student-2.jpg, student-3.jpg, student-4.jpg

Keep file names simple. Update every image alt text so visitors using screen readers understand the image.
